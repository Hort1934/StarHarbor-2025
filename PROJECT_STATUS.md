## 🎮 **HOW TO RUN:**

### Method 1: Docker (Recommended)
```bash
docker-compose up -d
# Frontend: http://localhost:3000
# API: http://localhost:8000
```

### Method 2: Local Development
```bash
# Terminal 1: API
python scripts/run_server.py

# Terminal 2: Frontend  
cd frontend && python -m http.server 3000
```
